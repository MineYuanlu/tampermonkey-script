__自动进行教学评价__  

打开教学评价界面，会自动选择"完全赞同"选项，但不会自动提交  
如果没有问题则可以一直点击提交，如果对部分老师有意见则可在提交前修改选项

如果在页面上看到“教学评价自动填充已加载 - yuanlu.”则代表脚本正常工作，否则代表脚本有问题
