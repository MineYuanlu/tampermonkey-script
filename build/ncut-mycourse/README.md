__自动完成平安毓秀学习（打开课程列表，全自动完成）__

**但无法自动考试**

2023.10更新: 无法自动完成验证码，请时刻观察

脚本会自动观看学习列表中所有动画，全部观看完毕时不会进行任何操作。
